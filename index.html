Dissertation Project


Facial recognition systems in law enforcement with decision making feedback.


The brief for this project was to identify and potentially solve a real-world problem. I identified the use of facial recognition technology and its inconsistent performances in real world scenarios as a project idea. My artefact was designed to perform real time facial recognition and provide match feedback to the user. The system created was able to identify faces from live video and compare that information to images from a pre-defined dataset. The system used advanced computer vision algorithms and techniques and was able to find and informing the user of any matching data. Key technology and languages used were OpenCV to handle computer vision and image recognition this was programmed using the Python language.
